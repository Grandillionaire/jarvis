It resists the attacks that owned the others

Self-hosted AI agents got compromised in production in 2026 — OpenClaw shipped a one-click RCE (CVE-2026-25253), had 20–42k gateways found exposed, and a skill registry that carried ~20% malware. Urfael was built blast-radius-first against exactly these. The difference isn't an adjective — it's a command:

npm run security

It boots the real daemon and dashboard and attacks them the way the wild did, then prints a pass/fail table. Latest run: 7/7 real-world attack classes resisted · 33/33 checks.

What it is

One brain, every surfaceA desktop Console, an ambient orb HUD, a full-screen TUI, a token-gated web dashboard, a CLI, and 8 owner-allowlisted chat channels — all thin clients of one local daemon.
It heard you, locallyPush-to-talk or a wake word; whisper.cpp transcribes and your machine speaks — no cloud STT/TTS by default.
Memory that compoundsEvery conversation distills into durable memory and a model of you, re-read each session; recall ranks its own history with BM25.
Quietly proactiveReminders, scheduled agent jobs (run the brain on a cron and deliver the result), and an opt-in heartbeat that stays silent unless something needs you.
Skills, installed paranoidIt learns reusable procedures — and any skill you install is previewed, safety-scanned, and never executed.
Open it upAn OpenAI-compatible API so Open WebUI / LibreChat / the openai SDK can drive it — locked to loopback.

Install

You need: a Claude Code subscription, signed in. macOS is primary; Linux is newer. No API key.

git clone https://github.com/Grandillionaire/urfael.git && cd urfael
./install.sh        # checks deps, scaffolds your vault — no keys
cd app && npm start # the Console opens

Or run it 100% on your own GPU — a local model (Ollama / NVIDIA NIM) + local voice means nothing leaves the machine. Guide.

It runs entirely on your own claude login — no credentials bundled or pooled, for ordinary individual use. Full guide: docs/SETUP.md.

What's lightly tested

This section exists on purpose. Honesty is a feature.

  • Every feature is verified end-to-end by an in-repo harness (npm run e2e) plus 75 unit tests, several adversarial security regressions.
  • Not yet exercised against real accounts: the live relay of the Matrix, Signal, and WhatsApp bridges (their logic is unit-tested). Code-complete and reviewed, not battle-hardened.
  • Linux is newer than macOS, and real-world scale is small — this is a personal tool, honestly stated. That's the one thing only time and users add.
